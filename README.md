# ConStrat AI

> Eliminating AI memory loss, preventing redundancy, and building topic authority across all channels with [Hindsight vector memory](https://github.com/vectorize-io/hindsight) and the Odysseus Orchestrator.

---

## 🚀 Overview
ConStrat AI is a memory‑powered content strategist agent.  
It helps creators and engineers:
- Draft and optimize posts with AI suggestions.
- Publish blogs with analytics and engagement tracking.
- Recall drafts, comments, and performance data using Hindsight.

---

## 🧩 Features
- **My Content Page** → Private workspace with AI editor, grammar/tone checks, trending feed, draft manager, and smart scheduler.  
- **My Blog Page** → Public showcase with analytics, comments, performance graphs, and export options.  
- **Agent Memory** → Powered by [Vectorize Agent Memory](https://vectorize.io/what-is-agent-memory), ensuring persistence and recall.  
- **Gamified Insights** → Badges for consistency, engagement, and trend spotting.  

---

## ⚙️ Tech Stack
- **Frontend**: React + TailwindCSS  
- **Backend**: Node.js / Express  
- **Memory Layer**: [Hindsight](https://hindsight.vectorize.io/)  
- **Deployment**: Vercel / Netlify (optional)
